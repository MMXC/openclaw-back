## BMAD-METHOD 研究

### 核心概念
- BMAD = Breakthrough Method for Agile AI-Driven Development
- AI Agent 编排框架，为敏捷团队每个角色提供专业化 Agent
- 两大创新：
  1. Agentic Planning - 专用 Agent 协作创建 PRD 和架构文档
  2. Context-Engineered Development - SM 将计划转化为超详细开发故事

### Agent 角色
| Agent | 角色 |
|-------|------|
| analyst | 业务分析师 |
| pm | 产品经理 |
| architect | 架构师 |
| dev | 开发者 |
| qa | QA 专家 |
| ux-expert | UX 设计师 |
| po | 产品负责人 |
| sm | Scrum Master |

### 工作流程
1. 规划阶段 (Web UI): Idea → Analyst → PM → Architect → PO 核对
2. 准备阶段: PO 将文档分片(shard)成小块故事
3. 开发阶段 (IDE): SM → Dev → QA → 完成 → 下一故事

### 安装命令
```bash
npx bmad-method install
```
