# VibeX 领域拆解示例

> 测试用例：在线教育平台

**用户需求：** "我想做一个在线教育平台，老师可以上传课程，学生可以购买学习"

---

## 拆解结果

### 总结
- 总领域数：8
- 核心域：3
- 支撑域：2
- 通用域：2
- 关联域：1

### 核心域

#### 1. 课程中心 (core_1)
- **优先级**：P0
- **描述**：课程管理与学习
- **子域**：
  - 课程管理（P0）：创建课程、编辑、上下架
  - 课程分类（P0）：学科、年级、难度
  - 课程详情（P0）：章节、课时、视频
  - 课程搜索（P0）：关键词、筛选
- **实体**：[课程, 章节, 课时, 分类]

#### 2. 学习中心 (core_2)
- **优先级**：P0
- **描述**：学生学习核心流程
- **子域**：
  - 视频学习（P0）：播放、进度记录
  - 学习记录（P0）：观看历史、学习时长
  - 课后练习（P0）：选择题、填空题
  - 课程评价（P1）：评分、评论
- **实体**：[学习记录, 练习题, 评价]

#### 3. 交易中心 (core_3)
- **优先级**：P0
- **描述**：付费购买
- **子域**：
  - 课程购买（P0）：支付、订单
  - VIP会员（P1）：会员卡、权益
  - 优惠券（P1）：兑换码、满减券
- **实体**：[订单, 会员, 优惠券]

### 支撑域

#### 4. 教师中心 (support_1)
- **优先级**：P1
- **描述**：老师教学管理
- **子域**：
  - 教师入驻（P1）：资质认证
  - 课程上传（P0）：视频上传、课后资料
  - 学生管理（P1）：查看学生、答疑
  - 收入管理（P1）：收益明细、提现
- **实体**：[教师, 收入记录]

#### 5. 直播授课 (support_2)
- **优先级**：P1
- **描述**：直播教学
- **子域**：
  - 直播教室（P1）：创建直播、邀请学生
  - 实时互动（P1）：弹幕、连麦
  - 直播回放（P2）：自动录制、回放
- **实体**：[直播, 回放]

### 通用域

#### 6. 用户中心 (generic_1)
- **优先级**：P0
- **描述**：用户体系
- **子域**：
  - 注册登录（P0）
  - 个人资料（P1）
  - 学习成就（P2）：勋章、证书
- **实体**：[用户, 成就]

#### 7. 消息通知 (generic_2)
- **优先级**：P1
- **描述**：通知推送
- **子域**：
  - 课程提醒（P1）：开课通知
  - 系统消息（P1）
- **实体**：[通知]

### 关联域

#### 8. 社区问答 (related_1)
- **关联方式**：扩展
- **发现原因**：学习平台常见补充功能

---

### 依赖关系图

```
课程中心(core_1) ← 依赖 ← 学习中心(core_2)
课程中心(core_1) ← 依赖 ← 交易核心(core_3)
学习中心(core_2) ← 依赖 ← 教师中心(support_1)
```
