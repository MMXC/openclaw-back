# 页面原型列表页 - 页面控件说明

## 1. 页面级说明

### 1.1 骨架结构

```
┌─────────────────────────────────────────────────────┐
│ Header: 返回 + 标题(3个页面) + [筛选] [+ 添加页面]   │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│  │ 页面1   │  │ 页面2   │  │ 页面3   │            │
│  │ 预览   │  │ 预览   │  │ 预览   │            │
│  │ 名称   │  │ 名称   │  │ 名称   │            │
│  │ 描述   │  │ 描述   │  │ 描述   │            │
│  │ 元素数 │  │ 元素数 │  │ 元素数 │            │
│  │ 操作   │  │ 操作   │  │ 操作   │            │
│  └─────────┘  └─────────┘  └─────────┘            │
│                                                     │
│  ┌─────────┐                                        │
│  │ + 新建  │                                        │
│  └─────────┘                                        │
└─────────────────────────────────────────────────────┘
```

### 1.2 布局

- Grid: 3列, gap 24px
- 卡片圆角: 12px
- 预览区高度: 180px

---

## 2. 控件级

### 2.1 页面卡片

| 控件 | 数据 | 交互 |
|------|------|------|
| 预览区 | 渐变背景 + 类型图标 | click → 编辑页 |
| 页面标签 | 右上角 badge | - |
| 名称 | page.name | - |
| 描述 | page.description | - |
| 元素数 | page.elementCount | - |
| 编辑 | icon-btn | click → 编辑 |
| 复制 | icon-btn | click → 复制 |
| 删除 | icon-btn | click → 确认删除 |

### 2.2 添加卡片

- 虚线边框
- hover: 边框变实 + 主题色

---

## 3. 状态

| 状态 | 说明 |
|------|------|
| default | 正常 |
| hover | 卡片上浮 |
| confirm-delete | 删除确认弹窗 |
