# 用户中心 - 页面控件说明

## 1. 页面级说明

### 1.1 页面骨架结构

```
┌─────────────────────────────────────────────────────┐
│ Header (64px)                                       │
│ [Logo] [搜索框] [通知] [头像]                        │
├──────────┬──────────────────────────────────────────┤
│ Sidebar  │ Main Content                             │
│ (200px)  │                                          │
│          │ ┌────────────────────────────────────┐   │
│ 我的项目 │ │ Welcome 横幅                        │   │
│ 模板市场 │ └────────────────────────────────────┘   │
│ 收藏夹   │ ┌────────────────────────────────────┐   │
│ 设置     │ │ 统计卡片 (3列)                      │   │
│ 帮助     │ └────────────────────────────────────┘   │
│          │ ┌────────────────────────────────────┐   │
│          │ │ 快捷操作 (新建/模板/导入)           │   │
│          │ └────────────────────────────────────┘   │
│          │ ┌────────────────────────────────────┐   │
│          │ │ 最近项目 (卡片网格)                │   │
│          │ └────────────────────────────────────┘   │
└──────────┴──────────────────────────────────────────┘
```

### 1.2 布局参数

| 区域 | 宽度 | 高度 |
|------|------|------|
| Header | 100% | 64px (固定) |
| Sidebar | 200px | calc(100vh - 64px) |
| Main | calc(100% - 200px) | 自适应 |

### 1.3 全局 CSS 样式

```css
.layout {
  display: flex;
  padding-top: 64px;
}

.sidebar {
  width: 200px;
  position: fixed;
  top: 64px;
  bottom: 0;
  border-right: 1px solid #e8e8e8;
}

.main {
  flex: 1;
  margin-left: 200px;
  padding: 32px;
}

.projects-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}
```

## 2. 控件级说明

### 2.1 Header 组件

| 控件 | 类型 | 交互 | 数据 |
|------|------|------|------|
| Logo | 文字 | click → 首页 | 固定 |
| 搜索框 | input | focus: 边框高亮 | placeholder: "搜索项目..." |
| 通知图标 | icon-btn | click → 通知列表 | 固定 |
| 用户头像 | avatar | click → 用户菜单 | 展示首字母 |

### 2.2 Sidebar 导航

| 控件 | 类型 | 交互 | 数据 |
|------|------|------|------|
| 我的项目 | menu-item | click → 列表页 | active 状态 |
| 模板市场 | menu-item | click → 模板页 | - |
| 收藏夹 | menu-item | click → 收藏页 | - |
| 设置 | menu-item | click → 设置页 | - |
| 帮助 | menu-item | click → 帮助页 | - |

**选中态**: 背景 #e6f7ff, 左边框 3px #1890ff

### 2.3 统计卡片

| 控件 | 数据 |
|------|------|
| 项目总数 | 用户项目 count |
| 本月创建 | 当月新建 count |
| 最近7天访问 | 7天内访问 count |

### 2.4 项目卡片

| 控件 | 类型 | 交互 | 数据 |
|------|------|------|------|
| 封面 | 背景图/渐变 | hover: 上浮 | 项目类型图标 |
| 名称 | 文字 | - | project.name |
| 描述 | 文字 | - | project.description |
| 更新时间 | 文字 | - | relative time |
| 操作菜单 | icon | click: 展开菜单 | 编辑/复制/删除 |

## 4. 页面状态

| 状态 | 说明 |
|------|------|
| default | 正常列表 |
| hover | 卡片/按钮悬停 |
| active | 当前菜单选中 |
| mobile | 移动端 (<768px) 侧边栏隐藏 |
