# 页面原型编辑页 - 页面控件说明

## 1. 页面骨架

```
┌─────────────────────────────────────────────────────────────────┐
│ Header: 返回 + 名称 + [撤销/重做] + [预览] [保存]               │
├──────────┬────────────────────────────────────┬───────────────┤
│ 组件面板  │ Canvas                              │ 属性面板      │
│ (200px)  │                                    │ (280px)       │
│          │  [桌面端|平板|移动端] 切换            │               │
│ 基础组件  │ ┌────────────────────────────────┐ │ 选中元素      │
│ - 按钮   │ │                                │ │ - 类型        │
│ - 输入框  │ │     画布编辑区域               │ │ - 尺寸        │
│ - 图片   │ │     (拖拽组件)                 │ │ - 样式        │
│          │ │                                │ │               │
│ 布局组件  │ └────────────────────────────────┘ │ 页面属性      │
│ - 容器   │                                    │ - 标题        │
│ - 栅格   │                                    │ - 背景        │
│          │                                    │               │
│ 表单组件  │                                    │               │
│ 业务组件  │                                    │               │
└──────────┴────────────────────────────────────┴───────────────┘
```

## 2. 控件说明

| 控件 | 交互 | 数据 |
|------|------|------|
| 组件项 | drag → 画布 | 添加 element |
| 画布元素 | click → 选中 | element.id |
| 元素拖拽 | move → 位置 | x, y |
| 设备切换 | click → 切换 | device: desktop/tablet/mobile |
| 属性输入 | blur → 更新 | element.props |

## 3. 状态

- default: 空闲
- dragging: 拖拽中
- selected: 元素选中
- editing: 编辑属性
