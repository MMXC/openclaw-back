# 登录/注册页 - 页面控件说明

## 1. 页面级说明

### 1.1 页面骨架结构

```
┌─────────────────────────────────────────┐
│         居中卡片 (400px宽)               │
│  ┌───────────────────────────────────┐  │
│  │ Logo + 副标题                      │  │
│  ├───────────────────────────────────┤  │
│  │ [登录] [注册] Tab 切换            │  │
│  ├───────────────────────────────────┤  │
│  │ 第三方登录按钮 (微信/钉钉)         │  │
│  ├───────────────────────────────────┤  │
│  │ 分隔符 "或"                        │  │
│  ├───────────────────────────────────┤  │
│  │ 表单 (邮箱/密码)                   │  │
│  │ [登录按钮]                         │  │
│  ├───────────────────────────────────┤  │
│  │ 忘记密码 · 立即注册 链接           │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

### 1.2 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| Desktop | ≥768px | 居中卡片 400px |
| Mobile | <768px | 全宽 padding 16px |

## 2. 控件级说明

### 2.1 Tab 切换

| 控件 | 类型 | 交互 | 数据 |
|------|------|------|------|
| 登录 Tab | 文字 | click: 切换到登录态 | active 状态 |
| 注册 Tab | 文字 | click: 切换到注册态 | - |

**交互**: 点击切换 active class，底部边框高亮

### 2.2 第三方登录

| 控件 | 类型 | 交互 | 数据 |
|------|------|------|------|
| 微信登录 | 按钮 | hover: 上浮+阴影 | 颜色: #07C160 |
| 钉钉登录 | 按钮 | hover: 上浮+阴影 | 颜色: #1488F5 |

### 2.3 表单控件

| 控件 | 类型 | 状态 | 验证 |
|------|------|------|------|
| 邮箱输入 | input[type=email] | default/focus/error | 必填、邮箱格式 |
| 密码输入 | input[password] | default/focus/error | 必填、最少6位 |
| 登录按钮 | button | default/hover/disabled | - |

**注册页额外控件**:
| 控件 | 类型 | 验证 |
|------|------|------|
| 昵称输入 | input | 必填 |
| 确认密码 | input | 与密码一致 |
| 服务条款 | checkbox | 必选 |
