## 使用方式

### 开发模式
```bash
cd docs/vibex-playground
npm install
npm run dev
# 访问 http://localhost:3000/landing
```

### 修改控件
1. 找到对应控件目录：`controls/控件名/`
2. 修改 `.tsx` 或 `.css` 文件
3. 追加修改记录到 `README.md` 的 **修改历史记录** 章节

---

## 骨架配置

文件: `skeleton.ts`

```typescript
{
  layout: 'stack',
  controls: [
    { id: 'header', type: 'Header', position: { order: 1 }, ... },
    { id: 'hero', type: 'Hero', position: { order: 2 }, ... },
    ...
  ]
}
```

### 拖拽修改后记录

1. 修改控件 props → 更新 `skeleton.ts` 中的 `controls[].props`
2. 调整控件位置 → 更新 `controls[].position`
3. 添加新控件 → 使用 `addControl()` 函数
4. 删除控件 → 使用 `removeControl()` 函数
