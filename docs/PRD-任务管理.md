# 任务管理应用 PRD

## 1. 产品概述

| 属性 | 内容 |
|------|------|
| **产品名称** | 任务管理 (Task Management) |
| **产品类型** | Web 应用 / 移动端适配 |
| **核心功能** | 任务的创建、编辑、删除、分配、跟踪、提醒 |
| **目标用户** | 个人用户、团队管理者、项目成员 |
| **产品阶段** | MVP |

---

## 2. 核心功能需求

### 2.1 任务管理 (CRUD)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建任务 | 支持标题、描述、截止日期、优先级、标签 | P0 |
| 编辑任务 | 修改任务所有字段 | P0 |
| 删除任务 | 软删除 + 回收站 | P1 |
| 任务详情 | 查看任务完整信息 | P0 |
| 任务列表 | 分页/无限滚动展示 | P0 |

### 2.2 任务组织

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 项目/看板 | 按项目/看板分组 | P0 |
| 标签 | 自定义标签筛选 | P1 |
| 优先级 | 高/中/低三级 | P0 |
| 截止日期 | 日期选择器 | P0 |

### 2.3 团队协作

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 成员管理 | 添加/移除成员 | P1 |
| 任务分配 | 指派给团队成员 | P1 |
| 任务评论 | 讨论任务细节 | P2 |

### 2.4 提醒通知

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 截止提醒 | 截止前提醒 | P1 |
| 指派提醒 | 被指派时通知 | P1 |

---

## 3. 页面结构

### 3.1 页面清单

| 页面 | 路径 | 描述 |
|------|------|------|
| 登录/注册 | `/login`, `/register` | 用户认证 |
| 首页/仪表盘 | `/` | 概览、快捷入口 |
| 项目列表 | `/projects` | 所有项目 |
| 项目详情 | `/projects/:id` | 项目下任务 |
| 看板视图 | `/projects/:id/board` | 看板模式 |
| 任务详情 | `/tasks/:id` | 任务详情页 |
| 个人中心 | `/profile` | 用户设置 |

### 3.2 页面交互流程

```
首页 ──┬── 项目列表 ── 项目详情 ── 看板视图
       │                      │
       │                      └── 任务详情
       │
       └── 个人中心 ── 我的任务
```

## 4. 数据模型

### 4.1 核心实体

#### User (用户)
```json
{
  "id": "string",
  "username": "string",
  "email": "string",
  "avatar": "string",
  "createdAt": "datetime"
}
```

#### Project (项目)
```json
{
  "id": "string",
  "name": "string",
  "description": "string",
  "ownerId": "string",
  "members": ["userId"],
  "createdAt": "datetime"
}
```

#### Task (任务)
```json
{
  "id": "string",
  "title": "string",
  "description": "string",
  "status": "todo|in_progress|done",
  "priority": "high|medium|low",
  "projectId": "string",
  "assigneeId": "string",
  "dueDate": "datetime",
  "tags": ["string"],
  "createdAt": "datetime",
  "updatedAt": "datetime"
}
```

## 7. 验收标准

### 功能验收
- [ ] 用户可以注册账号并登录
- [ ] 用户可以创建新项目
- [ ] 用户可以在项目中创建任务
- [ ] 用户可以编辑任务标题、描述、状态
- [ ] 用户可以删除任务
- [ ] 任务列表正确展示项目下所有任务

### 视觉验收
- [ ] 界面简洁现代
- [ ] 响应式布局（桌面端）
- [ ] 关键操作有交互反馈
